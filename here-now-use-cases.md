### 1. **Quick Shareables**
- **Ad-hoc dashboards** — Financial summaries, portfolio snapshots, KPIs
- **Meeting recaps** — Auto-formatted notes with action items
- **Data visualizations** — Charts, graphs, reports from analysis sessions
- **Proposal previews** — One-page pitch decks to share before formal docs

### 2. **AI Collaboration Artifacts**
- **Prompt libraries** — Curated lists of effective prompts
- **Agent outputs** — Quick HTML renders from generated content

### 3. **Client Touchpoints**
- **Personalized landing pages** — Custom per-client microsites
- **Gift guides** — Family office curated recommendations
- **Event pages** — Invitations, RSVPs, details (no Calendly needed)
- **Document portfolios** — Secure-ish shared galleries (time-boxed)

### 4. **Internal Tools**
- **Process documentation** — SOPs that actually get read
- **Training modules** — Interactive guides for new workflows
- **Checklist apps** — Simple HTML forms for recurring tasks
- **Resource directories** — Links, contacts, vendor lists

### 5. **Experiments & Tests**
- **A/B test pages** — Quick landing page variants
- **Form prototypes** — Test new intake flows
- **Widget demos** — Embed-able tools for the main site
- **Mock client portals** — Visualize before building

## Technical Notes

- **Anonymous publishes:** 24h expiry, 5/hour rate limit (currently maxed from launch traffic)
- **Authenticated:** Permanent, 60/hour, 5GB file limit
- **Custom domains:** Not supported (use Vercel for that)
- **Passwords/auth:** Roll your own or skip for low-stakes shares
