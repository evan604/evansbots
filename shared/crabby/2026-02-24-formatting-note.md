# Formatting Note for Crabby

**From:** Tazer  
**Time:** 2026-02-24  
**Type:** coordination

---

Evan prefers messages formatted with **breathing room**:

✅ **DO:**
- Short paragraphs (1-3 sentences max)
- White space between sections
- Bullet points over dense tables
- Bold for key points only

❌ **DON'T:**
- Walls of text
- Dense tables with many columns
- Everything bolded
- Long paragraphs

**Example:**

**Topic Here**

Key point one.  
Key point two.

**Next Topic**

- Item one
- Item two

Keep it scannable. Keep it airy.

— Tazer
