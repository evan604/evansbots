## Recommendation (mine)

**React + Tailwind** with [TailAdmin](https://github.com/TailAdmin/free-react-tailwind-admin-dashboard) free template.

**Why:**
- "Not ugly" guaranteed (full customization)
- Bot API built exactly for our needs (Tazer↔Crabby)
- Flat cost, no per-user pricing
- Skills we can reuse (React/Tailwind)

**Tradeoff:** 1-2 weeks vs 1-2 days. But Airtable's rate limits and UI boundaries will frustrate Evan eventually.
