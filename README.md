## What's Here

- `SOUL.md` / `IDENTITY.md` — Who Tazer is
- - `USER.md` — About Evan (preferences, goals, context)
  - - `AGENTS.md` — How to behave every session (read this first)
    - - `MEMORY.md` — Tazer's long-term curated memory (main session only)

        - - `tasks/` — Active task lists
          - - `skills/` — Tool and capability definitions
            - - `topics/` — Research topics and blog ideas
              - - `scripts/` — Utility scripts
                - - `shared/` — Shared assets (blog index, templates)
